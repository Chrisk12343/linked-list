# LinkedList Browser History

A simple C# console application demonstrating a doubly linked list implementation for browser history navigation.

## Project Overview

This project models browser history as a linked list of websites. It supports:

- displaying the full website list
- moving forward and backward through history
- adding a new website entry
- deleting an existing website
- searching for a website by name

The project includes a pre-loaded list of popular websites and a text-based menu for interaction.

## Technologies

- .NET 8.0
- C#

## Running the Project

### Prerequisites

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)

### Build and run

From the repository root:

```bash
dotnet run --project LinkedList.csproj
```

## Usage

When the application starts, it shows a menu with options to:

1. Display Website List
2. Go Forward
3. Go Backward
4. Add Website
5. Delete Website
6. Find Website
7. Exit

Enter the number for the action you want to perform.
